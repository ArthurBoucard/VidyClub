# Install vuejs.cli
```
npm install -g @vue/cli
```

## Project setup
```
npm install
```

### Add vuetify
```
vue add vuetify
-> choose default version
vue add router
--> no history
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
