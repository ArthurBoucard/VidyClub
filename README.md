# vidyclub

## <img width="26px" src="https://newsroom.ionis-group.com/wp-content/uploads/2018/12/epitech-logo-signature-quadri.png"/> Epitech project :

Website prototype made for a Jam in a week-end with using Vue.js

---

### :bulb: Idea :

The goal of VidyClub is to facilitate meetings and the sharing of passions. Each user has the ability to create "rooms" on the subject they want with the support of videos from any video hosting site such as YouTube for example. After creating a room, users can connect and communicate through text or voice chat.

---

### :gear: Technologies :

• Vue.js
• Vue CLI
• Vuetify
